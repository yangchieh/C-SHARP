## XmlSchemaParser

**用途**：  
從指定的 XML 檔案讀取「系統 / 類別 / 欄位 / 欄位值」結構，並在主控台以對齊格式輸出資料欄位資訊，用於瀏覽或檢查系統資料結構定義。

---

## 執行環境

- **.NET**：.NET 6 / .NET 7（視你的專案實際設定而定）
- **OS**：Windows / Linux / macOS 皆可

---

## 專案結構

- `Program.cs`
  - 進入點 `Main`
  - 解析 XML 並輸出結構資訊
  - 包含字串對齊輔助函式 `PadRightChinese`

---

## XML 結構假設

程式假設 XML 具有以下層級結構（節點名稱固定）：

- 根節點：`SYSTEMS`
  - 屬性：
    - `NAME`：主系統名稱
- 子節點：多個 `SYSTEM`
  - 屬性：
    - `NAME`：系統模組名稱
  - 內含多個 `CLASS`
    - 屬性：
      - `NAME`：類別代號
      - `TITLE`：類別名稱 / 說明
    - 內含多個 `FIELD`
      - 屬性：
        - `NAME`：欄位代號
        - `TITLE`：欄位名稱 / 說明
        - `TYPE`：欄位型別（例：`CHAR`、`NUMBER`…）
        - `SIZE`：欄位長度（例：`10`、`50`…）
      - 子節點（選用）：多個 `VALUE`
        - 屬性：
          - `NAME`：代碼值
          - `TITLE`：代碼說明

> 若實際 XML 結構不同，請調整程式對應的節點名稱與屬性名稱。
